# Smart ERP Next

**Hệ thống quản trị doanh nghiệp thế hệ mới** — vượt trội ERPNext, Odoo, KiotViet, Nhanhvn, MISA về tốc độ, trải nghiệm và khả năng mở rộng.

## Tại sao chọn Smart ERP Next?

| Tính năng | Smart ERP Next | KiotViet | Nhanhvn | MISA | ERPNext |
|-----------|:-:|:-:|:-:|:-:|:-:|
| Native đa nền tảng (Web + Mobile + Desktop) | ✅ | ❌ | ❌ | ❌ | ❌ |
| Offline-first + CRDT sync | ✅ | ❌ | ❌ | ❌ | ❌ |
| Mã nguồn mở | ✅ | ❌ | ❌ | ❌ | ✅ |
| Bản địa hóa tiếng Việt built-in | ✅ | ✅ | ✅ | ✅ | Partial |
| Multi-tenant | ✅ | ❌ | ❌ | ❌ | ✅ |
| Real-time WebSocket | ✅ | Partial | Partial | ❌ | ❌ |
| Monorepo shared packages | ✅ | ❌ | ❌ | ❌ | ❌ |
| POS tích hợp | ✅ | ✅ | ✅ | ❌ | ✅ |

## Modules

- 🏠 **Dashboard** — Tổng quan doanh thu, đơn hàng, tồn kho, insights
- 🛒 **POS** — Bán hàng tại quầy với 6 phương thức thanh toán
- 📦 **Sản phẩm** — CRUD, SKU, tồn kho, danh mục phân cấp
- 🏭 **Kho hàng** — Nhập/xuất/điều chỉnh kho, lịch sử giao dịch
- 👥 **Khách hàng** — CRM, công nợ, điểm tích lũy, nhóm KH
- 🚚 **Nhà cung cấp** — Quản lý NCC, công nợ, ngân hàng
- 📋 **Mua hàng** — Đơn nhập hàng, nhận hàng từng phần
- 🛍️ **Đơn hàng** — Quản lý đơn bán, state machine, thanh toán
- 📊 **Báo cáo** — Doanh thu, lợi nhuận, tồn kho, khách hàng
- ⚙️ **Cài đặt** — Công ty, ngôn ngữ, thông báo, bảo mật

## Tech Stack

| Thành phần | Công nghệ |
|-----------|-----------|
| Monorepo | pnpm + Turborepo |
| Backend | NestJS 10, Drizzle ORM, PostgreSQL 14+ |
| Web | Next.js 15, React 19, Tailwind CSS |
| Mobile | Expo 52, React Native 0.76 |
| Desktop | Tauri 2 (Rust + WebView) |
| i18n | i18next (vi/en built-in) |
| Offline | Dexie (IndexedDB) + CRDT vector clocks |
| Real-time | Socket.IO |

## Bắt đầu nhanh

```bash
git clone https://github.com/smart-erp/smart-erp-next.git
cd smart-erp-next
pnpm install

# Cấu hình môi trường
cp apps/api/.env.example apps/api/.env

# Chạy database migration
psql $DATABASE_URL -f packages/database/drizzle/0001_initial_schema.sql

# Khởi động tất cả services
pnpm dev
```

Xem thêm tại [Hướng dẫn phát triển](/docs/development).
